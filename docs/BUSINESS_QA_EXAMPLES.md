# Business QA Examples

This file provides representative user question patterns for chatbot retrieval.

## Inventory Questions

- What is the current inventory status?
- How many products are in stock?
- How many lenses are used?
- Show unmatched lenses summary.
- Does serial number `ABC123` exist in inventory?

## Invoice Questions

- Show details for invoice `9140481167`.
- Which invoice contains serial number `SN0001`?
- List problematic invoices.
- Which invoices are unmatched?
- How many unmatched invoice rows are there?

## Supplier and Company Questions

- How many suppliers are in the system?
- How many companies or manufacturers are configured?
- Show supplier overview.

## Site / Clinic Questions

- How many clinics are configured?
- Show inventory by site.
- Which clinic has this serial number?

## System Usage / Documentation Questions

- How do I start the backend service?
- How do I run the frontend?
- How can I fix database connection refused?
- How to upload invoice PDF and learn new format?

## Clarification Examples

- Unmatched invoice rows are not the same as unmatched invoice numbers.
- In-stock count is `total - used`.
- Matched/unmatched belongs to reconciliation status, not only stock status.
