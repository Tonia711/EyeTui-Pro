# User Quick Tasks

This page gives short action guides for common user tasks.

## Task: Receive new lenses quickly

1. Open **Lens Receiving**.
2. Select clinic/site.
3. Scan barcode or upload Excel.
4. Confirm serial numbers were saved.

## Task: Mark lens usage from surgery file

1. Open **Lens Usage & Invoice Reconciliation**.
2. Upload usage Excel.
3. Verify parsed rows.
4. Submit update.

## Task: Reconcile invoice PDF

1. Open **Lens Usage & Invoice Reconciliation**.
2. Upload invoice PDF.
3. Review extracted invoice number, supplier, serial numbers.
4. Use **Edit/Learn** if needed.
5. Save and check unmatched summary.

## Task: Find where a lens is

1. Open **Lens Inventory**.
2. Search by serial number.
3. Check site/clinic and status.

## Task: Check problematic invoices

1. Open **Invoice List** to browse and filter.
2. Use chatbot question: “Which invoices are problematic?”
3. Review returned invoice numbers and unmatched counts.

## Task: Add missing master data

1. Open **Settings**.
2. Add supplier/company/lens type/site.
3. Retry the failed receiving or reconciliation step.

## Task: Export reconciliation output

1. Go to reconciliation page.
2. Run or refresh reconciliation.
3. Use export action to download results.
